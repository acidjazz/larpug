# Requirements

* [NodeJS v6.x](https://nodejs.org/en/)
* [npm v3.x](https://www.npmjs.com/)


# Installation

Require this package with Composer

```bash
composer require acidjazz/larpug
```

Install the needed node modules to run pug
```bash
cd vendor/acidjazz/larpug/node
npm install
```
